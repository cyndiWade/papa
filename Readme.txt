1. 用户登录，注册 （新浪微博，腾讯QQ登录）
	用户类型 ：1）监理师。（授权监理的项目列表，项目主题添加，预约列表）
	2）客户（业主），设计师等
	
2. 拍拍说
	图片+语音+文字形式的主题，图片为必需。
	回复不能是图片，只能语音和文字。
	所有人可回复，回复需先登录。
	
	1）图片语音主题发布 （拍拍说，项目）
		选择接收人（站内推荐，好友列表）顺便发送微博注册邀请
		接收人获得一个动态（问答）通知
	
	2）主题列表 主题+最新3条回复 [记录最新3条回复的id]
	3）主题详细内容页 主题+所有回复
	4）动态主题列表 （我发表的主题有新的回复，被指定为接收人的主题发表），动态数量提示
	5）我参与过的主题列表（回复过或发表的主题）
	
3.	照片式监理
	所有项目列表，每个帐号对应一个项目
	按钮 ：创建项目/管理我的项目
	
	1）创建项目，授权监理师图片语音主题添加权限（选监理师后自动添加到已授权列表）
	2）编辑项目：选择施工阶段，添加项目图片语音主题(不需要选接受人)
	3）项目详细内容（业主和授权监理师可添加主题，同编辑项目）
	4）项目列表
	
4.选监理师
	1）监理师列表
	2）申请预约[填写相关预约表单信息] （监理师同意后，自动拥有客户项目的图片语音主题添加权限）
	
5. 好友功能
	1）添加好友
	2）删除好友
	3）好友列表
	4）邀请微博好友注册
	
	
数据库表：
		1.用户表（监理师）
		2.拍拍说主题表，项目主题表
		3.评论表，项目主题评论表
		4.项目表 用户id
		5.好友关系表
		6.预约表
		7.监理师拥有的项目表 监理师user_id 项目id
 		8.动态表 拍拍说待接收主题，主题有新的回复 关系表 主题id，接收人， （已读 时间）
		9.文件表（图片，声音）
